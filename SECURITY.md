# Security policy

## Supported versions

| Version | Supported |
| --- | --- |
| 0.3.x | Yes |
| Earlier versions | No |

## Reporting a vulnerability

Please use [GitHub private vulnerability reporting](https://github.com/gamjak/codex_cost_extension/security/advisories/new) for security issues. Do not open a public issue or include sensitive logs, tokens, workspace paths, or proof-of-concept details in a public discussion.

We aim to acknowledge private reports within five business days. An acknowledgement is not a promise of a fix, disclosure date, or severity assessment; we will coordinate next steps privately.
